# 🌦️ Weather App

A modern and responsive Weather App built using **HTML, CSS, and JavaScript**. It fetches real-time weather data from the **OpenWeatherMap API** and displays weather information for any city entered by the user.

## 🚀 Features

- 🔍 Search weather by city name
- 🌡️ Real-time temperature
- 💧 Humidity information
- 💨 Wind speed
- 🤗 Feels like temperature
- 📊 Atmospheric pressure
- 🌤️ Weather condition
- 🖼️ Dynamic weather icon
- 🌙 Dark / Light Mode
- ⌨️ Search using Enter key
- ❌ Invalid city error handling
- 📱 Responsive Design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeatherMap API

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/weather-app.git
```

2. Open the project folder

```bash
cd weather-app
```

3. Open `index.html` in your browser.

## 🔑 API Setup

1. Create an account on OpenWeatherMap.
2. Generate your API key.
3. Replace your API key inside `script.js`.

```javascript
const apiKey = "871f5162ebfb9a44f78881863d484c56";
```
